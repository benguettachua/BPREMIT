#README

1st Step
Create a sqlite3 database within the same directory as the python file.
Run the following command in terminal to create the database named ‘BPREMIT’:
sqlite3 /'YOUR DIRECTORY'/BPREMIT

Run the sql script to create a data table within the database itself.

2nd Step
Install the relevant python libraries to make sure the python file runs as per normal.

3rd Step
Run the python script and it will start to populate the database and table.

*BPREMIT database file is a sample database that is loaded with the current data already.
